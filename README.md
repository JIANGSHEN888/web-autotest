# Web Autotest Project

> Project of UI-Autotest,which applied to Collection system, has combined with Selenium, TestNG and Allure

> 此UI自动化工程用于催收系统，已集成Selenium、TestNG和Allure等框架组件。

````text
hyxf.AutoTest.webtest
├── Collection (催收业务)
│   ├── example (样例)
│   ├── page (页面与元素)
│   └── testcase (测试用例)
│
└── Utils (web测试常用工具)
````
